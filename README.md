# 📖 Car-expense-tracker <a name="about-project"></a>

**Car-expense-tracker** keeps track of all your expenses related to your car. It will allow you to keep track of road taxes, insurance and maintenance costs.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- CSS
- HTML
- Ruby 
- Ruby on Rails
- Postgresql

<!-- Features -->

### Key Features <a name="key-features"></a>

- **calculate fuel mpg**
- **set maintenance dates**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need Ruby and  Postgresql installed on your computer.

### Setup

Clone this repository to your desired folder: 

`git clone https://github.com/vflorin1995/Car-exoense-tracker.git`


### Install

Then install the project's necessary dependencies with: 

`bundle install`
